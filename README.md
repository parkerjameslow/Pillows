# Pillows

A simple, mobile-friendly checklist for tracking the duties of setting up a boutique show. Multiple people can view and update the same list in real time.

## How it works

- Open the website on your phone or computer.
- Add tasks, check them off, or remove them.
- Everyone looking at the site sees the same list, updated live.

## Tech

- Plain HTML, CSS, and JavaScript (no build step).
- [Firebase Firestore](https://firebase.google.com/docs/firestore) for shared real-time data.
- Hosted on GitHub Pages or Firebase Hosting.
